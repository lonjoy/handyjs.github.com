Handy

2012-11-22

Handy 是由支付宝前端团队创建的基于 SeaJS 和 spm 的 wap 模块精选集，她能帮助你快速创建 wap 站点。

如何使用

模块的使用和开发请参见通用教程：

  • 快速入门：Hello SeaJS: 5分钟上手教程
  • 进阶教程：Hello spm: 使用 spm 和 SeaJS 开发一个中型项目

wap 开发过程中特别需要注意的地方请参见 wap 开发指南

模块精选

CSS

  • handy.base-ui@1.0.0

基础

  • arale.events@1.0.0
  • arale.class@1.0.0
  • arale.base@1.0.1
  • arale.widget@1.0.2

工具

  • gallery.zepto@1.0.0

UI

  • handy.switchable@1.0.0
  • arale.validator@0.8.9
  • gallery.iscroll@4.1.9

我们不做什么

  • 不做模拟的dialog，使用系统自带的alert、confirm 和 prompt。
  • 不做模拟的滑动换页效果。
